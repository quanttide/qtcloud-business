# 产品需求文档

量潮销售云（qtcloud-sales）产品需求文档，聚焦报价与合同管理系统设计。

## 产品定位

销售云是 B2B 销售场景下的后端工具，核心定位于：

- **报价制定** - 方案生成、报价单生成
- **合同管理** - 合同模板、签约流程

→ [查看 BRD](../brd/index.md)

## 核心能力

### 1. 报价制定

帮助商务经理高效制定报价方案。

- 支持方案模板（公司简介、方案PPT）
- 支持报价单生成（产品明细、价格、折扣）
- 报价历史记录与版本管理

### 2. 合同管理

帮助商务经理管理合同流程。

- 合同模板库
- 合同签署流程（发送、签署、归档）
- 合同履约跟踪

## 用户故事

### 报价

**US1**: 作为商务经理，我希望根据模板快速生成报价单，以便减少准备时间。

- 接受标准：
  - 能选择方案模板并填充客户信息
  - 能调整产品明细和价格
  - 能导出报价单 PDF

**US2**: 作为商务经理，我希望查看报价历史，以便参考历史报价。

- 接受标准：
  - 能按客户搜索历史报价
  - 能查看报价版本

### 合同

**US3**: 作为商务经理，我希望使用合同模板快速生成合同，以便减少起草时间。

- 接受标准：
  - 能选择合同模板并填充客户信息
  - 能预览合同内容

**US4**: 作为商务经理，我希望跟踪合同签署进度，以便及时催促。

- 接受标准：
  - 能查看合同状态（待签署/已签署/已归档）
  - 能收到签署提醒通知

## 核心指标

| 指标 | 说明 |
|------|------|
| **报价生成效率** | 从模板到生成的平均耗时 |
| **合同签署率** | 发送后签署成功的比例 |
| **平均签单周期** | 从报价到签署的平均天数 |

## 技术约束

- **数据安全**：HTTPS/TLS 1.3 传输，电子合同加密存储
- **多端支持**：Web、iOS、Android 三端数据实时同步
