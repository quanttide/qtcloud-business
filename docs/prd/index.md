# 产品需求文档

量潮商务云（qtcloud-business）产品需求文档，聚焦获客与客户管理系统设计。

## 产品定位

商务云是 B2B 销售场景下的获客工具，帮助商务经理获取潜在客户、建立信任。

→ [查看 BRD](../brd/index.md)

## 核心能力

### 1. 获客渠道

帮助商务经理将案例信息发布到多个渠道，获得更多潜在客户咨询。

- 案例一键多渠道发布
- 发布效果统计

### 2. 信任背书

帮助商务经理用真实案例建立信任。

- 案例库管理
- 案例自动脱敏

### 3. 潜客归集

帮助商务经理归集和管理潜客信息。

- 各渠道潜客自动归集
- 潜客跟进记录

## 用户故事

### 获客渠道

**US1**: 作为商务经理，我希望将案例发布到多个渠道，以便获得更多潜客咨询。

- 接受标准：能录入案例并选择渠道发布

**US2**: 作为商务经理，我希望查看发布效果，以便优化策略。

- 接受标准：能查看曝光量、咨询量

### 信任背书

**US3**: 作为商务经理，我希望展示案例给客户，以便建立信任。

- 接受标准：能搜索和展示案例

### 潜客管理

**US4**: 作为商务经理，我希望归集潜客信息，以便集中管理。

- 接受标准：能查看潜客来源、需求

**US5**: 作为商务经理，我希望记录跟进情况，以便避免遗漏。

- 接受标准：能添加沟通记录

## 核心指标

| 指标 | 说明 |
|------|------|
| 案例曝光量 | 累计曝光次数 |
| 咨询转化率 | 曝光到咨询的比例 |
| 潜客跟进率 | 24h内跟进的比例 |
