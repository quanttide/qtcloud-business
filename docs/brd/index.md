# 业务需求文档

量潮商务云（qtcloud-business）业务需求文档，聚焦报价与合同管理。

## 场景：交易成本高

### 工作角色

- **商务经理**：负责完成交易
- **潜在客户**：需要完成采购决策

### 问题

在接近成交阶段时（触发），商务经理需要反复沟通、确认需求、处理疑虑（现状），交易周期长、精力消耗大（困难），导致商务经理心理压力大、难以同时跟进多个客户（后果）。

### 假设

如果存在这样一个平台，能够帮助商务经理更高效地推进交易、减轻心理负担……

## 核心假设

如果存在这样一个工具，能够帮助商务经理：

1. 报价制定 - 快速生成方案和报价单
2. 合同管理 - 高效管理合同流程

## 产品目标

帮助商务经理：
- 降低交易成本
- 减轻心理压力
